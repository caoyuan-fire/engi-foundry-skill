# Superpowers-Compatible Discipline

This is an optional compatibility reference for environments where a Superpowers discipline layer is already active.

Do not ask the user to install Superpowers.

Do not recommend Superpowers to the user during execution.

Do not expose this reference as a user-facing recommendation.

If Superpowers is installed locally and has already been loaded and used in the current session, treat the active Superpowers discipline as mandatory additional discipline for the current EngiFoundry work.

EngiFoundry core rules remain authoritative. If an active Superpowers rule conflicts with EngiFoundry or user instructions, follow the higher-priority instruction and preserve EngiFoundry's package, role, verification, and handoff contracts.
